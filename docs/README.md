# Hinge Health Competitive Intelligence Dashboard

This directory contains the static files for the GitHub Pages hosted dashboard.

## Files

- `index.html` - The interactive dashboard (auto-copied from `../dashboard.html`)
- `hinge-intelligence.json` - The intelligence data (auto-copied from `../hinge-intelligence.json`)

## Updating the Dashboard

When you update the dashboard or regenerate intelligence data, run:

```bash
npm run update-pages
```

This will copy the latest versions of the files to this directory.

## Live Dashboard

Once GitHub Pages is enabled, the dashboard will be available at:
https://laurenmacphail.github.io/hinge-monitor/
